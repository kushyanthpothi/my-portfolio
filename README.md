## 📋 Overview

Kushyanth Portfolio is a high-performance web application built with **Next.js 16.1.1** and **React 19.2.3**, leveraging the **Firebase** ecosystem for scalable backend services. Designed with a professional glassmorphism aesthetic, it features advanced animations, real-time content management, and AI-driven content generation. This project serves as a comprehensive digital identity, integrating SEO best practices and specialized performance optimizations.

---

## ✨ Key Features

### 🎯 Core Capabilities

- **🚀 Modern Architecture**: Engineered with Next.js 16.1.1 App Router and React 19.2.3.
- **📝 Content Management**: Dynamic blog and project system powered by Firestore with real-time updates.
- **🤖 AI Integration**: Automated blog drafting using Google Gemini AI, with multi-modal support for OpenAI and OpenRouter.
- **🛠️ Professional Admin Panel**: Secure CMS at `/admin` for centralized content and analytics management.
- **🎨 Visual Excellence**: Premium UI featuring glassmorphism, fluid gradients, and silky-smooth animations.
- **🌓 Adaptive Theming**: Intelligent Dark/Light mode synchronization with system preferences and persistent storage.
- **⚡ SEO & Performance**: Fully optimized with JSON-LD structured data, dynamic sitemaps, and robots.txt.
- **📱 Responsive Design**: Seamless experience across mobile, tablet, and ultra-wide monitor resolutions.
- **🎬 Interactive UX**: Integration of Framer Motion, Lenis smooth scrolling, and custom magnetic interactions.

---

## 🏗️ Tech Stack

### Frontend & UI
- **Framework**: Next.js 16.1.1 (App Router)
- **Library**: React 19.2.3
- **Styling**: Vanilla CSS, CSS Modules, Tailwind CSS 4.x
- **Animations**: Framer Motion, Lenis Scroll, GSAP
- **Typography**: Antonio (Headings), Inter (Body)

### Backend & Cloud
- **Database**: Firebase Firestore
- **Auth**: Firebase Authentication (Admin Access)
- **Storage**: Firebase Cloud Storage (Media)
- **Functions**: Firebase Cloud Functions (Server-side Logic)

### AI & Automation
- **Providers**: Google Gemini, OpenAI, OpenRouter
- **Utilities**: EmailJS for contact delivery

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ & npm
- Firebase Project setup

### Installation
1. `git clone https://github.com/kushyanthpothi/my-portfolio.git`
2. `npm install`
3. Configure `.env.local` with your credentials.
4. `npm run dev`
